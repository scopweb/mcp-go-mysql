## Características principales

<CardGrid stagger>
  <Card title="10 herramientas" icon="setting">
    Consultas, inspección de tablas, escrituras, análisis de planes de ejecución e información del servidor.
  </Card>
  <Card title="Clasificador de verbos" icon="siren">
    Lista blanca de verbos SQL aceptados. Gestión de privilegios, acceso al sistema de archivos y sentencias apiladas se rechazan antes de llegar al driver.
  </Card>
  <Card title="Umbral por filas" icon="rocket">
    Las escrituras que afectan a más de `MAX_SAFE_ROWS` filas requieren `confirm_key`. Cubre el "uy, se me olvidó el WHERE" sin falsos positivos.
  </Card>
  <Card title="Errores honestos" icon="approve-check">
    Los errores del driver pasan tal cual para que el LLM pueda corregirse. Sin teatro de sanitización.
  </Card>
</CardGrid>

<CardGrid>
